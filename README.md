# SOC Security Incident Investigations

## Overview

This repository demonstrates practical SOC analyst workflows including
alert triage, log analysis, and incident documentation.

The scenarios simulate real security events that require investigation
and response by security analysts.

## Incident Types Investigated

- Phishing email alerts
- Suspicious login activity
- Malware detection alerts

## Skills Demonstrated

- Security alert triage
- Log analysis
- Threat identification
- Incident documentation
- Security reporting

## Tools & Concepts

- SIEM alert investigation
- Python log parsing
- Incident response playbooks
